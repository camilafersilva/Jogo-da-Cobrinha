
# Jogo da Cobrinha

Este é um simples jogo da cobrinha desenvolvido em JavaScript, HTML e CSS.


## Como Jogar

1. Abra o jogo no seu navegador.
2. Use as setas do teclado para controlar a cobrinha.
3. Colete as "comidas" vermelhas para crescer a cobra.
4. Evite colidir com as bordas do jogo ou com você mesmo, caso contrário, o jogo terminará.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Pré-requisitos

Não são necessários pré-requisitos especiais para jogar o jogo no seu navegador.
